# 자동차 게임

백개길 자동차 게임 미션 저장소

# 구현해야할 기능 목록
- [X] 이름 입력 (최소 1대 이상)
    - [] ',' 기준 이름 분리
        - [] ERROR : ',' 존재하지 않을 경우
    - [] 각 이름 5자 이하
        - [] ERROR : 5자 초과
        - [] ERROR : 공백 또는 빈 문자
    
- [] 시도 회수 입력
    - [] 숫자 입력
        - [] ERROR : 숫자가 아닌 문자 입력
        - [] ERROR : 공백 또는 빈 문자
        
- [] 자동차 전진
    - [] 0 ~ 9 random 값을 가져오는 기능
    - [] 4 이상일 경우에만 전진

- [] 중간 진행상황 출력
    - [] 각 자동차의 현재 위치 출력

- [] 결과 출력
    - [] 가장 멀리 이동한 자동차들의 이름 출력
